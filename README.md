### Ejecución
1.  **Compilar los archivos de TypeScript:**
    ```bash
    tsc saludo.ts operaciones.ts
    ```

2.  **Ejecutar los ejercicios en la terminal:**
    ```bash
    node operaciones.js
    ```

3.  **Ver el ejercicio web:**
    Abrir el archivo `index.html` en el navegador.
